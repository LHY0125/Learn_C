# C语言学习代码库

## 项目简介
这是一个完整的C语言学习代码库，包含了从基础语法到高级项目的全面学习内容。代码库涵盖了翁凯老师C语言课程的所有章节练习、课堂代码实践以及完整的项目实战。

## 技术栈

- **编程语言**：C语言（C99标准）
- **编译器**：GCC
- **开发环境**：支持Windows/Linux/macOS
- **特殊库**：数学库（-lm）

## 编译说明

### 单文件编译
```bash
gcc filename.c -o output
```

### 需要数学库的程序
```bash
gcc filename.c -o output -lm
```

### 多文件项目（如五子棋）
```bash
cd 课上代码练习/五子棋
gcc "五子棋 copy 3.c" -o gomoku -lm
```

### 🔰 初学者路径
1. **基础语法**：从翁凯C语言第1-3章开始

3. **数组掌握**：重点练习第5章数组操作
4. **指针突破**：深入理解第6章指针概念

### 💡 高级应用

2. **动态内存**：第12章可变数组
3. **程序架构**：第9章大型程序设计

## 使用说明

1. **克隆仓库**：
   ```bash
   git clone https://github.com/LHY0125/Learn_C.git
   cd Learn_C
   ```

2. **选择学习内容**：
   - 基础学习：进入`翁凯C语言/`对应章节
   - 项目实战：进入`课上代码练习/五子棋/`
   - 快速练习：查看`课上代码练习/`基础练习

3. **编译运行**：
   ```bash
   gcc 文件名.c -o 输出名
   ./输出名
   ```
